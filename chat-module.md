# Документация модуля Chat (ManagerFix)

Модуль чата управляет форматом сообщений, локальным и глобальным чатом, личными сообщениями (ЛС), звуками, тултипами при наведении и связанными командами.

## Обзор возможностей

- **Формат чата** — MiniMessage для локального и глобального сообщений (бейджи, цвета).
- **Локальный чат** — сообщения видят только игроки в радиусе (настраивается); префикс `!` — отправка в глобальный чат.
- **Личные сообщения (ЛС)** — `/pm`, `/tell`, `/msg`, быстрый ответ `/r`.
- **Блокировка и игнор** — `/pmblock`, `/ignore` (кто не может писать в ЛС).
- **Spy** — ChatSpy (локальные сообщения вне радиуса + ЛС), CommandSpy (команды игроков).
- **Звуки** — при отправке/получении ЛС и при сообщениях в локальном чате (вкл/выкл в конфиге).
- **Тултипы** — при наведении на ник: баланс (Vault), «ЛКМ — личное сообщение»; при наведении на текст сообщения: время отправки, ПКМ — копировать текст.
- **Join/Quit/Death** — свои форматы сообщений (опционально).

## Команды

| Команда | Описание | Пример |
|--------|----------|--------|
| `/chattoggle` | Переключение режима чата (локальный / глобальный) | `/chattoggle` |
| `/chatspy` | Вкл/выкл просмотр локальных сообщений вне радиуса и ЛС | `/chatspy` |
| `/commandspy` | Вкл/выкл просмотр команд других игроков | `/commandspy` |
| `/pm <игрок> <сообщение>` | Личное сообщение игроку | `/pm Steve Привет` |
| `/tell` | То же, что `/pm` | `/tell Steve Привет` |
| `/msg` | То же, что `/pm` | `/msg Steve Привет` |
| `/r <сообщение>` | Быстрый ответ последнему, кто писал в ЛС (или кому вы писали) | `/r Ок, договорились` |
| `/pmblock <игрок>` | Запретить/разрешить игроку писать вам в ЛС | `/pmblock Steve` |
| `/ignore add <игрок>` | Добавить игрока в чёрный список (он не сможет писать вам в ЛС) | `/ignore add Steve` |
| `/ignore remove <игрок>` | Убрать из чёрного списка | `/ignore remove Steve` |
| `/ignore list` | Список игроков в чёрном списке | `/ignore list` |

## Конфигурация chat.yml

Файл: `plugins/ManagerFix/modules/chat.yml` (или в jar: `modules/chat.yml`).

### Основные форматы чата

| Параметр | Описание | Плейсхолдеры |
|----------|----------|--------------|
| `format-local` | Формат сообщения в локальном чате (в радиусе) | `{badge}`, `{player}`, `{message}`, `{prefix}`, `{suffix}` |
| `format-global` | Формат сообщения в глобальном чате | те же |
| `badge-local` | Бейдж для локального сообщения (например ｢𝐋｣) | — |
| `badge-global` | Бейдж для глобального (например ｢𝐆｣) | — |

### Локальный чат

| Параметр | Описание | Значение по умолчанию |
|----------|----------|------------------------|
| `local-radius` | Радиус видимости локальных сообщений (блоки). `0` = только глобальный чат | `60` |
| `local-chat-sounds-enabled` | Включить звуки при отправке/получении сообщений в локальном чате | `true` |
| `local-sound-send` | Звук при отправке сообщения в локальный чат (имя из `org.bukkit.Sound`). Пусто или `none` — без звука | `ENTITY_EXPERIENCE_ORB_PICKUP` |
| `local-sound-receive` | Звук при получении сообщения из локального чата | `BLOCK_NOTE_BLOCK_HAT` |

Поведение: сообщение без префикса `!` уходит в локальный чат (видят игроки в радиусе); с префиксом `!` — в глобальный (видят все).

### Спам и глобальный чат

| Параметр | Описание | Значение по умолчанию |
|----------|----------|------------------------|
| `spam-cooldown` | Кулдаун между сообщениями (сек). `0` = отключён | `2` |
| `global-sent-note-enabled` | Показывать отправителю подсказку после отправки в глобальный чат (если другой плагин ругается на чат) | `false` |

### Личные сообщения (ЛС)

| Параметр | Описание | Значение по умолчанию |
|----------|----------|------------------------|
| `badge-pm` | Бейдж в ЛС (например ｢𝐏𝐌｣) | ｢𝐏𝐌｣ |
| `format-pm` | Формат строки ЛС | `{badge}`, `{sender}`, `{receiver}` |
| `pm-sounds-enabled` | Включить звуки при отправке/получении ЛС | `true` |
| `pm-sound-send` | Звук при отправке ЛС | `ENTITY_EXPERIENCE_ORB_PICKUP` |
| `pm-sound-receive` | Звук при получении ЛС | `BLOCK_NOTE_BLOCK_PLING` |

### Тултип при наведении на ник

| Параметр | Описание | Значение по умолчанию |
|----------|----------|------------------------|
| `hover-enabled` | Показывать тултип при наведении на ник в чате (баланс, «ЛКМ — личное сообщение») | `true` |
| `hover-format` | Текст тултипа (MiniMessage) | Плейсхолдеры: `{player}`, `{balance}` |

### Тултип при наведении на текст сообщения

| Параметр | Описание | Значение по умолчанию |
|----------|----------|------------------------|
| `message-hover-enabled` | Показывать время отправки при наведении на текст сообщения; ПКМ — копировать текст | `true` |
| `message-hover-format` | Текст тултипа (MiniMessage) | Плейсхолдер: `{time}` |
| `message-hover-time-format` | Формат времени (Java `DateTimeFormatter`) | `HH:mm` |

### Прочее

| Параметр | Описание | Значение по умолчанию |
|----------|----------|------------------------|
| `colors-enabled` | Разрешить игрокам цвета в чате (без отдельного права) | `true` |

### Join / Quit / Death

| Параметр | Описание | Плейсхолдеры |
|----------|----------|--------------|
| `format-join` | Сообщение при входе игрока. Пусто = ванильное | `{player}` |
| `format-quit` | Сообщение при выходе | `{player}` |
| `format-death` | Сообщение о смерти. Ванильное сообщение о смерти не показывается | `{player}` |

## Форматы и плейсхолдеры

Во всех форматах поддерживается **MiniMessage** (в т.ч. `<#hex>`, `<gray>`, `<white>` и т.д.). В конфиге можно использовать и коды Minecraft (`§a`, `§c` и т.п.).

### Чатовые форматы (format-local, format-global)

- `{badge}` — бейдж (｢𝐋｣ или ｢𝐆｣).
- `{player}` — имя отправителя.
- `{message}` — текст сообщения.
- `{prefix}`, `{suffix}` — из Vault (если плагин и Vault установлены).

### Формат ЛС (format-pm)

- `{badge}` — бейдж ЛС.
- `{sender}` — отправитель (для получателя показывается «Вы» или имя).
- `{receiver}` — получатель (для отправителя показывается «Вы» или имя).

### Тултип по нику (hover-format)

- `{player}` — имя игрока.
- `{balance}` — баланс из Vault (если есть).

### Тултип по сообщению (message-hover-format)

- `{time}` — время отправки в формате `message-hover-time-format`.

## Локальный и глобальный чат

1. **Локальный чат** (по умолчанию):
   - Игрок пишет сообщение **без** `!` в начале.
   - Сообщение видят только игроки в радиусе `local-radius` (и при включённом ChatSpy — шпионы).
   - Формат берётся из `format-local`, бейдж из `badge-local`.
   - При включённых звуках: отправителю — `local-sound-send`, получателям в радиусе — `local-sound-receive`.

2. **Глобальный чат**:
   - Игрок пишет сообщение **с префиксом `!`** (например `!Всем привет`).
   - Текст после `!` — само сообщение; его видят все в онлайне.
   - Формат — `format-global`, бейдж — `badge-global`.
   - При `global-sent-note-enabled: true` отправителю может отправляться подсказка после отправки.

3. **ChatSpy** (`/chatspy`):
   - Игроки с правом и включённым ChatSpy видят локальные сообщения **вне** радиуса и личные сообщения (ЛС) с префиксом [Spy] / [PM Spy].

## Личные сообщения (ЛС)

- **Отправка:** `/pm <игрок> <сообщение>`, `/tell`, `/msg` — одно и то же.
- **Быстрый ответ:** `/r <сообщение>` — ответ последнему собеседнику по ЛС (кто писал вам или кому вы писали).
- **Ограничения:**
  - Нельзя писать самому себе.
  - Если вас заблокировали через `/pmblock`, вы не сможете писать этому игроку.
  - Если вы в игноре у получателя (`/ignore add`), вы не сможете ему писать.
- **Звуки:** при `pm-sounds-enabled: true` играют `pm-sound-send` (отправителю) и `pm-sound-receive` (получателю). Отключение: `pm-sounds-enabled: false` или пустые/`none` у звуков.

---

## Звуки

- **ЛС:** ключи `pm-sounds-enabled`, `pm-sound-send`, `pm-sound-receive`. Имена звуков — из [org.bukkit.Sound](https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/Sound.html) (например `BLOCK_NOTE_BLOCK_PLING`, `ENTITY_EXPERIENCE_ORB_PICKUP`). Пусто или `none` — звук не играет.
- **Локальный чат:** ключи `local-chat-sounds-enabled`, `local-sound-send`, `local-sound-receive`. Логика та же: вкл/выкл и имена звуков.

## Тултипы (hover)

1. **На ник в чате** (при `hover-enabled: true`):
   - Показывается текст из `hover-format` (баланс, «Нажмите ЛКМ — личное сообщение»).
   - ЛКМ по нику подставляет в чат `/pm <ник> `.

2. **На текст сообщения** (при `message-hover-enabled: true`):
   - Показывается время отправки и подсказка про копирование (из `message-hover-format`).
   - ПКМ по тексту сообщения копирует **только текст сообщения** в буфер обмена.

---

## Join / Quit / Death

- Если в `chat.yml` заданы непустые `format-join`, `format-quit`, `format-death`, они используются вместо ванильных сообщений.
- Для смерти ванильное сообщение о причине смерти не показывается; показывается только строка по `format-death` с плейсхолдером `{player}`.

---

## Интеграции

- **Vault:** при установленном Vault используются `{prefix}`, `{suffix}` в форматах чата и баланс в тултипе по нику.
- **PlaceholderAPI:** плейсхолдеры в сообщениях и форматах обрабатываются через PlaceholderAPI, если плагин установлен.
- **GoodMod:** при замьюченности игрока (GoodMod) он не может писать в чат; модуль чата проверяет мьют перед отправкой.
